# Seriefolkets stora pris

[Seriefolkets stora pris](https://seriefolkets-stora-pris.vercel.app/)

**SERIEFOLKETS STORA PRIS** instiftades 2024. Medlemmarna i Facebook-gruppen [Forum för seriefolk](https://www.facebook.com/groups/seriefolk "Forum för seriefolk på Facebook") nominerar och röstar fram årets serievinnare i ett antal kategorier. Priset består av äran, folkets kärlek och ett diplom. Du kan läsa mer om Facebook-gruppen på [Seriewikin.](https://seriewikin.serieframjandet.se/index.php/Forum_för_seriefolk "Seriewikin om Forum för seriefolk")

En liten hemsida bygggd med Hugo och temat Blowfish.
